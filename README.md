## Description
LinkinPurry is a Single Page web based application, created using Fastify and Typescript for the server-side and Tailwind, React for client-side. This Application is made to fulfill the project requirement of Web Application Development (IF3110) Course. This application utilizes docker for containerization, PostgreSQL for RDBMS.

## Requirement
docker
fastify
node.js v20

## How to Install
1. Install Requirement in this URL: https://www.docker.com/get-started
2. Clone this repository

## How to Run Server
1. Make sure you are in the root directory of the repository
2. Fill each variable in .env file.
3. Open terminal and run this:
```
docker-compose --env-file ./backend/.env up --build
```
4. Open frontend folder
5. Open terminal and run this:
```
npm run dev
```
